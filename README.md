# Noosphere

> *The sphere of human thought — distributed, open, and owned by no one.*
> *人類の知性の圏 — 分散され、開かれ、誰のものでもない。*

---

## What is Noosphere? / Noosphereとは

AIは今、少数の巨大企業が独占しています。  
学習には膨大なGPUと資金が必要で、その負担は一部の企業だけが背負っています。

**Noosphereはその構造を変えようとするプロジェクトです。**

世界中の人々がブラウザ一つで参加し、それぞれの計算資源を少しずつ提供することで、  
誰もが使えるAIを、誰の負担にもならない形で育てていく。

人類の知識はインターネット上に広がっている。  
それを人類全員で学習させ、人類全員に返す。

---

## Vision / ビジョン

- **For everyone** — 特定の企業や国家に依存しない、公共インフラとしてのAI
- **By everyone** — ブラウザから参加できる、敷居ゼロの分散学習
- **Owned by no one** — 中央管理者なし、止められない、検閲されない
- **Sustainable** — 巨大企業の赤字構造ではなく、分散した小さな貢献の集積

---

## How it works / 仕組み

```
あなたのブラウザ  ─┐
他の誰かのPC      ─┼─→  分散学習ネットワーク  →  Noosphere AI
GPUを持つ誰か    ─┘
```

1. **参加する** — ブラウザを開くだけで計算リソースを提供できる
2. **貢献する** — データの前処理、トークナイズ、勾配計算を分担
3. **使う** — 学習されたモデルはネットワーク全体に返される
4. **広げる** — 参加者が増えるほど、システムは強くなる

## Why now? / なぜ今か

ChatGPT、Gemini、Claude——これらはすべて赤字です。  
巨大な計算コストを少数の企業が背負い続けることは持続可能ではありません。

同時に、世界中の何十億台ものデバイスが、  
毎日何時間もアイドル状態のままCPUとGPUを眠らせています。

**眠っている計算資源を、人類の知性のために使う。**  
それがNoosphereの答えです。

## How to contribute / 参加する方法

**エンジニア・研究者の方へ**  
Issues や Discussions で設計の議論に参加してください。  
特に以下の分野の知見を求めています：

- 暗号理論 / Byzantine耐性

このプロジェクトを広めることが最大の貢献です。  
ビジョンに共鳴したら、SNSでシェアしてください。
